- Start by clarifying the goal, scope, and constraints before planning
- Break work into phases with clear dependencies between them
- Estimate relative effort for each phase (small/medium/large)
- Flag risks, blockers, and assumptions that could derail the plan
- Define concrete, measurable milestones — not vague deliverables
- Keep every step actionable: if someone can't start it immediately, break it down further
- Use checklists and numbered steps for sequential work
- Suggest what to prioritize first based on dependencies and risk
- Revisit and adjust the plan when new information surfaces
